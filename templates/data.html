Data

To train a Machine Learning model for predicting the compressive strength, the dataset from the University of California, Irvine is used. Following is the summary of Concrete Compressive Strength Data Set:
Number of instances 1030
Number of independent features 8
Number of dependent features 1
Missing values No

List of features and its type:

Feature Type Unit Independent?
Cement Numerical Kg/Cu.M Yes
Blast Furnace Slag Numerical Kg/Cu.M Yes
Fly Ash Numerical Kg/Cu.M Yes
Water Numerical Kg/Cu.M Yes
Superplasticizer Numerical Kg/Cu.M Yes
Coarse Aggregate Numerical Kg/Cu.M Yes
Fine Aggregate Numerical Kg/Cu.M Yes
Age Numerical Days Yes
Concrete compressive strength Numerical MPa No
