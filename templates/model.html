Machine Learning Model

Linear Regression is a type of Machine Learning technique, which is used to predict the numerical outcome - whether discrete or continuous. The compressive strength of a concrete block is a non-linear function with respect to the ingredients it contains. As compressive strength is a numerical value (Load / Cross-sectional area), the linear regression technique is used for given ingredients.

Evaluation technique used:

Train: 80%; Test: 20%

Evaluation metrics used:

Root Mean Square Error (RMSE)

Hyperparameter tuning technique used:
